A knowledge Based Agent that navigates a Wumpus World grid
using propositional logic and Resolution Refutation.

## How to Run
1. Make sure Python 3 is installed
2. Run: python wumpus_agent.py
3. Open browser at: http://localhost:3000

## Features
- Dynamic grid size
- Random pit and Wumpus Placement each episode
- Propositional logic knowledge base (kB)
- CNF conversion and Resolution Refutation engine
- Step by step or Auto run mode
- Real time mertrics dashboard
- Green = safe, Gray = unknown, Red = Danger

## Tech Stack
- Python 3
- Vanilla HTML, CSS, Javascript fronted
- Built in http.server for web serving
